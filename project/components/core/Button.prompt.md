**Button** — primary action control; navy primary, teal accent, outline secondary, plus ghost & link. Use for any clickable action or CTA across ACHI surfaces.

```jsx
<Button variant="primary" size="md">Read the report</Button>
<Button variant="accent">Subscribe</Button>
<Button variant="secondary">Learn more</Button>
<Button variant="link" href="/publications">See more publications →</Button>
```

Variants: `primary` (navy), `accent` (teal), `secondary` (outline), `ghost`, `link`.
Sizes: `sm` · `md` · `lg`. Props: `block`, `iconLeft`, `iconRight`, `href` (renders `<a>`), `disabled`.
