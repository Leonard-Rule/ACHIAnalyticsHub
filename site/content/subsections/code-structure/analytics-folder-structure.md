## Folder Creation & Version Management
id: folder-standards
type: sop

For each project created, the primary analyst assigned to that project is responsible for creating folders on G:\ and R:\ if they have not yet been created.

- Each folder should contain a 'previous versions' subfolder to hold everything but the most current version
- All work (including works in progress) must be stored on G:\ or R:\
- Exception: working datasets may be stored on the SSDs
