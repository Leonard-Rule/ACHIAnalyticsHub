## NewID — member identifier built from submitter + member ID
id: newid-rule
type: reference

NewID is the combination of Submitter (ME001) + memberid (ME107), therefore it can be created for any claim. NewIDs are member-level identifiers in the APCD.

[callout warning "NewID counts are member counts, not individual counts"]
NewID is created by submitter, so the same person can have multiple NewIDs in the same year and can even have multiple NewIDs at the same time if they have dual coverage. This means any distinct NewID counts are called member counts but would never be a count of total individuals.
[/callout]
