## StudyID — de-identified person-level identifier
id: studyid-rule
type: reference

Last name (encrypted as ME998) + Sex + DOB. Also known as Hash_ID (just last name (encrypted as ME998) + DOB). Use cases: link HTI and HDI, follow people over time.

[callout warning "StudyIDs can be unreliable"]
Since StudyIDs are based on a person's last name, they can be unreliable because last names change when people get married, divorced, etc. They are still a handy identifier, but there will always be limitations when using StudyID.
[/callout]
