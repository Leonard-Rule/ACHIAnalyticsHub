## All projects follow a consistent numbered step layout
id: step-structure
type: sop

The 'what it should do' part will vary by project but this is a general guide for standard projects.

| Step | What it should do |
|---|---|
| Step 0 | Library setup, macro definitions, Excel imports — nothing analytical |
| Step 1 | Member / denominator population table creation |
| Step 2 | Claims pull of interest, cleaning |
| Step 3 | Event creation, flagging, filtering |
| Step 4 | Analytical output table creation |
